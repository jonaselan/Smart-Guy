Informações técnicas

Para que o jogar venha a executar é necessario a instalação do Ruby e o framework Gosu

Regras

regra do jogo, acrescentar o contexto

Download

Linux

$ sudo apt-get install ruby
$ sudo gem install gosu

Windows

http://rubyinstaller.org/

Gosu

https://github.com/jlnr/gosu/wiki

Para mais informações acesse:

https://www.ruby-lang.org/pt
http://libgosu.org

Como jogar

Colete todas as chaves e entre no seu apartamento, para isso utilize as teclas:

Move o personagem para esquerda

Move o personagem para direita

Salto do personagem
